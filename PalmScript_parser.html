PalmScript - Parser

Grammar

  • IMM -> *IMM | [0-9]+ | (IMM BOP IMM)
  • BOP -> + | - | * | / | % | < | > | =
  • LBL -> _[a-zA-Z0-9]+
  • CMD -> jnz IMM LBL | set IMM IMM | out IMM | in IMM | LBL:
  • S -> CMD S | EOF

Example Program - A+B

in 0
in 1
out (*0 + *1)

Example Program - Factorial

in 0
set 1 1

_loop:
jnz (*0 < 2) _exit
set 1 (*1 * *0)
set 0 (*0 - 1)
jnz 1 _loop

_exit:
out *1
